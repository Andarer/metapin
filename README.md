# MetaPIN

**MetaPersonalInfrastructureNode**

***Meta Personal Infrastructure Node***

Android как автономный инфраструктурный узел.

Приоритет локального подключения.
Возможность работы в автономном режиме.
Модульная конструкция.
Управление человеком.

---

## Vision

MetaPIN превращает устройства Android в портативные узлы инфраструктуры.

Устройство становится:
- локальный сервер
- узел точки доступа
- участник сети
- центр офлайн-экосистемы

---

## MVP Цели

- Управление точкой доступа
- Локальный портал
- Обнаружение устройств
- Обмен файлами
- Архитектура, ориентированная на работу в автономном режиме

## Философия

- Приоритет локального доступа
